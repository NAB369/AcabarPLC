# 🏦 Enterprise Banking Project Tech Stack

Here is the complete and up-to-date tech stack for the **AcabarPLC enterprise banking project**.

---

## 🎨 Frontend — User Interface

| Category | Technology |
|---|---|
| Framework | Next.js v16.2.4 |
| Core Library | React v19 |
| Language | TypeScript |
| Icons | Lucide React |
| Charts & Visuals | Recharts |
| 3D Graphics | Three.js, React Three Fiber |
| Image Processing | React Image Crop |
| Localization | Cambodia Gazetteer |

---

## ⚙️ Backend — API & Business Logic

| Category | Technology |
|---|---|
| Framework | NestJS v11 |
| Language | TypeScript |
| Database ORM | Prisma |
| Database Engine | SQLite with LibSQL adapter |
| Authentication | Passport.js with JWT |
| Security & Encryption | Argon2, Bcrypt |
| Data Validation | Class Validator, Class Transformer |
| Testing | Jest — Unit & E2E Testing |
| Scheduling | NestJS Schedule for cron jobs |

---

## 📌 Summary

This project uses a modern full-stack TypeScript architecture with:

- **Next.js** for the frontend
- **NestJS** for backend APIs
- **Prisma** for database management
- **JWT authentication** for secure access
- **Recharts and Three.js** for advanced data visualization
- **Jest** for reliable testing

---

## 🚀 Project Goal

The goal of this project is to build a secure, scalable, and modern enterprise banking platform with strong authentication, clean architecture, and interactive financial visualizations.
