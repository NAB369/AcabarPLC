Here is the complete and up-to-date Tech Stack for your enterprise banking project:
-----------------------------------------------------------------------------------
🎨 **Frontend (User Interface)**
Framework: Next.js (v16.2.4)
Core Library: React (v19)
Language: TypeScript
Icons: Lucide React
Charts & Visuals: Recharts
3D Graphics: Three.js & React Three Fiber (for advanced interactive visualizations)
Image Processing: React Image Crop
Localization: Cambodia Gazetteer (for local province/district data)

⚙️ **Backend (API & Business Logic)**
Framework: NestJS (v11)
Language: TypeScript
Database ORM: Prisma
Database Engine: SQLite (powered by LibSQL adapter)
Authentication: Passport.js with JWT (JSON Web Tokens)
Security & Encryption: Argon2 & Bcrypt (for password hashing)
Data Validation: Class Validator & Class Transformer
Testing: Jest (Unit & E2E Testing)
Scheduling: NestJS Schedule (for cron jobs)
